Easily set product stock to Zero when you see it, with one click!

== Description ==
 
This plugin is made for administrator to reset stock to zero for simple products using a simple button click on products in product shop page frontend. No need to search for product using sku or name just see product picture and click on its button below.
 
== Installation ==
 
1. Upload the plugin folder to your /wp-content/plugins/ folder.
1. Go to the **Plugins** page and activate the plugin.
 
== Frequently Asked Questions ==
 
= How do I use this plugin? =
 
Upload the plugin from your wordpress site Dashboard and just activate it. You can also upload it in the plugin folder using FTP.
look visualy for your product and confirm it with the indicated sku if defined already.
Then just click on reset button.
 
= How to uninstall the plugin? =
 
Simply deactivate and delete the plugin.
